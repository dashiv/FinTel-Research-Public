FinTel Research

AI-Powered Equity Intelligence Platform

FinTel automates institutional-grade investment research — reading SEC filings, scoring signals with ML models, and delivering actionable intelligence to investors in real time.

What We Do

  • Scans SEC EDGAR daily for new IPO filings (S-1 / F-1)
  • Reads full filing text using large language models (Claude Sonnet)
  • Extracts revenue model, burn rate, competitive moat and risk factors
  • Scores each company 0–100 across 12 market-regime ML models
  • Executes paper trades via Interactive Brokers API with guardrails
  • Delivers morning digest and instant alerts via Telegram

Technology

Python Claude API XGBoost IBKR TWS API Streamlit AWS PostgreSQL

Status
